There are three ways to connect CSS with html

  1. Inline CSS
    To style we need to add style attribute to each element. It will be rearly used.

  2. Internal CSS
    In this all the styling content is written in style tag which is within head tags.

  3. External CSS
    In this a .css file is creted and all the styling content is written in this file and finally link with html page.
